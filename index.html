Diffie-Hellman Key Exchange

Creating Shared Secrets Over Public Channels

The Challenge

How can Alice and Bob create a shared secret key when a hacker is listening to everything they say? The answer: Diffie-Hellman, based on the difficulty of solving discrete logarithms.

The Protocol
  1. Public Agreement: Alice and Bob publicly agree on a large prime number (q) and a primitive root (α)
  2. Private Keys: Alice chooses secret XA, Bob chooses secret XB
  3. Public Keys: Alice computes YA = αXA mod q and sends it publicly. Bob does the same with YB
  4. Shared Secret: Both compute K using the other's public key and their own private key. Amazingly, they get the same K!
